# Architecture Extension Proposals (AEPs)

*Process: [`../process/aep-process.md`](../process/aep-process.md) · Template: [`../templates/aep-template.md`](../templates/aep-template.md)*

AEPs define the **public extension contracts** third parties build against: planners, runtimes, compiler backends, the package format, channel adapters, and policy/knowledge/secret providers. Each AEP carries an explicit **stability level** (Experimental → Beta → Stable) and a conformance suite.
