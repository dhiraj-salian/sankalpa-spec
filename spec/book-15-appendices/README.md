# Book 15 — Appendices

*Status: Draft skeleton · Nature: Informative (reference material).*

## Scope
Cross-cutting reference material and end-to-end walkthroughs that tie the Books together. Non-normative, but essential for comprehension and onboarding.

## Chapters
1. **`01-end-to-end-walkthrough.md`** — One intent followed the whole way: Intent → Goals → Planning → High IR → Optimize → Policy → Low IR → Runtime selection → Execution → Events → Experience → Knowledge, with the actual artifacts at each step.
2. **`02-notation-reference.md`** — Consolidated notation: Resource schema, IR grammar, Event naming, diagram conventions.
3. **`03-worked-examples-catalog.md`** — A library of representative scenarios across runtimes and planners.
4. **`04-mapping-to-prior-art.md`** — Side-by-side: where a Sankalpa concept has an analog in Linux/LLVM/Kubernetes/Temporal/Git/Terraform, and where it diverges.
5. **`05-faq-and-anti-patterns.md`** — Common misunderstandings (e.g., "isn't this just an agent framework?") and the anti-patterns the architecture forbids.
6. **`06-glossary-mirror.md`** — Pointer to the authoritative [Glossary](../../GLOSSARY.md).
